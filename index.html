String hashing is the way to convert a string into an integer known as a hash of that string. An ideal hashing is the one in which there are minimum chances of collision (i.e 2 different strings having the same hash). Polynomial rolling hash function.In this hashing technique, the hash of a string is calculated as: hash = (s[0]*P^{0} + s[1]*P^{1} + ....s[m]*P^{m}) mod M Where P and M are some positive numbers. And s[0], s[1], s[2] … s[n-1] are the values assigned to each character in English alphabet (a->1, b->2, … z->26). 

                long long compute_hash(string const& s) {
                    const int p = 31;
                    const int m = 1e9 + 9;
                    long long hash_value = 0;
                    long long p_pow = 1;
                    for (char c : s) {
                        hash_value = (hash_value + (c - 'a' + 1) * p_pow) % m;
                        p_pow = (p_pow * p) % m;
                    }
                    return hash_value;
                }
